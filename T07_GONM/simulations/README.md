# Simulations

The GONM line also includes physical simulations, especially molecular optimization and coarse-grained folding demonstrations.

The scripts included here document:

- the progression from `LJ-12` proof of concept to `LJ-38` continuation, confinement, and escape-freeze refinement under CPU-only execution;
- a compact `LJ-13` atomic-structure calculation as a visual entry point to cluster optimization;
- a simple real-element calculation through an `Ar13` argon cluster with its own Lennard-Jones parameters;
- a quantitative-finance simulation in which GONM searches a sparse high-dimensional portfolio under a risk-return objective;
- a logistics and delivery-routing simulation in which GONM organizes geographic sectors and contracts multi-vehicle routes;
- a neural-hyperparameter simulation in which GONM crosses broad plateaus and enters a narrower validation basin than a local Adam-style baseline;
- a quantum variational simulation in which GONM deforms a wavefunction family toward a lower-energy ground-state candidate in an anharmonic double well;
- a relativistic geodesic simulation in which GONM searches a Schwarzschild trajectory with lower geodesic residual and better invariant preservation near the strong-field region;
- a reduced cosmological N-body simulation in which GONM keeps finite short-range separations while naive gravity rapidly approaches ultraviolet collapse;
- a quantum wavefunction optimization simulation in which GONM minimizes an effective many-electron variational functional while keeping the coefficients on a normalized physical manifold;
- a relativistic boundary-value simulation in which GONM optimizes the whole Schwarzschild path between fixed endpoints instead of integrating it step by step;
- a control-engineering simulation in which GONM tunes a feedback law for an inverted pendulum under strong wind gusts;
- an aerospace-control simulation in which GONM tunes a noisy-sensor attitude loop for satellite pointing under disturbance torques;
- a smart-grid simulation in which GONM redispatches generation and storage after a line outage to recover a safe operating point;
- a neural-compression simulation in which GONM prunes a compact hidden layer more effectively than a simple magnitude-based rule;
- a first visual coarse-grained `protein folding` demonstration driven by the same layered GONM logic.
