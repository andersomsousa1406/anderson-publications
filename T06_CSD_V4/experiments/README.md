# Experiments

The experimental support for `V4` is centered on the canonical multichart solver, the ontological transition criterion from `V3`, and the stability of the external gauge.

## Main scripts

- `v4_solver.py`
- `validate_v4_solver.py`
- `transition_vs_v3_validation.py`
- `calibrate_tau_k.py`
- `gauge_stability_validation.py`
- `saddle_guided_restart_validation.py`
- `compare_v2_v3_v4_current.py`

These scripts cover the main support lines discussed in the manuscript: solver validation, transition against `V3`, calibration of the canonical multichart count, gauge stability, and the current comparative benchmark across `V2`, `V3`, and `V4`.
