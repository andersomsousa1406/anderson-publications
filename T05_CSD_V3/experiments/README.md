# Experiments

The experimental support for `V3` is centered on comparisons against the affine regime, robustness checks for the nonlinear chart, and degraded observational validation.

## Main scripts

- `quadratic_functional.py`
- `robustness_suite.py`
- `compare_v2_v3_batch10_multiseed.py`
- `compare_v2_v3_v4_batch10_multiseed.py`
- `observational_validation.py`
- `gauge_comparison.py`

These scripts cover the main support lines discussed in the manuscript: classical multiseed comparison, robustness, observational degradation, and chart/gauge behavior.
